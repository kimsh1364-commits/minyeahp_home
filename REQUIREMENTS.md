## 결과 표시 방식
- **단일 추천**: 가장 많이 나온 성격 유형 하나만 표시
- 예: "당신은 대담한 모험가! 더블 에스프레소를 추천해요."

## 비주얼 스타일
- **스타일 1 기준**: 발랄하고 컬러풀
- 밝은 색상, 둥근 모양, 이모지 아이콘 포함
- 각 답변 옆에 이모지 아이콘 표시
- 배경: 그라디언트 (핑크-옐로우-민트)
- 카드: 흰 배경, 둥근 모서리, 그림자

## 이미지 & 아이콘
- ✅ 각 결과마다 커피 이미지 포함 (public/ 폴더에 저장됨)
- ✅ 각 답변 옆에 이모지 아이콘 포함

## 퀴즈 질문 (6개)

### Q1. 해리포터 세계에 살게 된다면 어느 기숙사?
- ⚡ 그리핀도르 (용감하고 대담) → **대담한 모험가**
- 🌿 허플퍼프 (친절하고 성실) → **달콤한 열정가**
- 💚 슬리데린 (야망있고 전략적) → **실용주의자**
- 💙 래번클로 (지혜롭고 건강함) → **건강 덕후**

### Q2. 넷플릭스 마라톤을 한다면?
- ⚔️ 긴장감 넘치는 액션 시리즈 → **대담한 모험가**
- 🌸 따뜻한 로맨틱 코미디 → **달콤한 열정가**
- 🧘 잔잔한 자연 다큐멘터리 → **건강 덕후**
- 🎯 일단 뭐든 틀고 딴짓하기 → **실용주의자**

### Q3. 이번 주말 계획은?
- 🏔️ 즉흥 등산 또는 새로운 액티비티 → **대담한 모험가**
- 🛁 집에서 홈카페 & 달콤한 디저트 → **달콤한 열정가**
- 🧗 요가 또는 러닝 → **건강 덕후**
- 🛋️ 밀린 할 일 처리 → **실용주의자**

### Q4. 마블 어벤져스 중 나와 가장 비슷한 캐릭터는?
- 🦸 아이언맨 (혁신적, 강렬) → **대담한 모험가**
- 🌟 스파이더맨 (따뜻하고 유쾌) → **달콤한 열정가**
- 🌿 블랙 위도우 (건강하고 자기관리 철저) → **건강 덕후**
- 🛡️ 캡틴 아메리카 (실용적, 책임감) → **실용주의자**

### Q5. 여행 스타일은?
- 🌋 무계획 배낭여행, 즉흥적으로 → **대담한 모험가**
- 🏖️ 리조트에서 여유롭게, 맛집 투어 → **달콤한 열정가**
- 🚴 자전거 여행 또는 하이킹 코스 → **건강 덕후**
- 📋 꼼꼼하게 계획 세우고 효율적으로 → **실용주의자**

### Q6. 아침에 일어나서 제일 먼저 하는 것은?
- 🏃 바로 운동 또는 찬물 샤워 → **대담한 모험가**
- 😴 스누즈 한 번 더, 천천히 일어나기 → **달콤한 열정가**
- 🧘 스트레칭 또는 명상 → **건강 덕후**
- 📱 오늘 할 일 체크하기 → **실용주의자**

## 기술 스택
- Next.js + JavaScript
- 별도 데이터베이스 없음 (클라이언트 사이드 로직)
- 결과 계산: 각 답변의 성격 유형 집계 → 최다 유형 표시
